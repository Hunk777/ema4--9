## v2.1 (2025-11-10) - 全期間EMA表示版 📊
**新機能: チャート全履歴にEMAライン表示**

### 変更内容
- ✅ **EMA全期間表示機能**
  - EA起動時点からではなく過去データ全体にEMAラインを描画
  - Fast/Slow/Trend の3本すべて対応
  - 表示バー数を調整可能（デフォルト500本）

- ✅ **Slow EMA専用ハンドル追加**
  - より正確な描画のため独立したインジケーター使用
  - 3本のEMAすべてが独立した計算

- ✅ **パラメータ追加**
  ```
  InpMaxBarsToShow = 500     // 表示する過去バー数
  InpShowSlowEMA = true      // Slow EMA個別表示制御
  ```

### メリット
- 📊 チャート全体でEMAの流れが把握可能
- 🎯 過去のクロスポイントも視覚的に確認
- ⚡ 初回描画後は新バーのみ更新で高速動作

### 推奨設定
```
InpTimeframe = PERIOD_M1
InpEMA_Fast = 5
InpEMA_Slow = 13
InpEMA_Trend = 34
InpMaxBarsToShow = 500     // チャート範囲に応じて調整
```

## v2.0 (2025-11-10) - 分足ベース版 🎯
**大きな変更: 秒足→分足ベースに完全移行**

  - より安定したシグナル

  - MT5組み込みEMA計算
  - 高速・正確
  - メモリ効率向上

- ✅ **時間軸選択可能**
  - M1（1分足）デフォルト
  - M5、M15、M30、H1対応

### メリット
- 📈 ノイズが大幅に減少
- 🎯 シグナル精度向上
- ⚡ 処理速度向上
